# prayertime-scraper
 scrapes data from www.islamicfinder.org/  _this is Experimental Project_
#### Installation
<pre><code>composer require sajidh/prayertime-scraper
</code></pre>

#### Usage
`use sajidh\prayerscraper\Prayer;

$scraper = new Prayer;
$scraper->collection();
echo $scraper->Sunrise;`
